# Sparets 
Single Page Application React Typescript 

## Features
- typescript
- yarn, webpack 2, build with dll
- react
- hot module reload
- express
- tslint, tslint-config-standard, editorconfig
- CSS modules
- TODO
    - postcss, cssnext
    - jest

    - redux

## Install
`yarn install`

## Run development  
`npm run build:dll`  
`npm start`
